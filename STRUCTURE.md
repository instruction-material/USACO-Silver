# USACO-Silver Structure

This repository is a problem bank for `USACO Silver`. Historical folder names
remain, but each active problem now uses the common wrapper shape.

Active public folders stay at the repository top level and contain
`README.md`, `starter/`, and `solution/`.

Some migrated problems have only one real legacy snapshot; the opposite role
folder contains a README placeholder until a distinct starter or solution is
authored.

New Silver material should use the wrapper shape and should not add unlinked
alternate versions beside active folders.
